サービス一覧

目的に応じて選べる3つのサービスをご用意しています。個人向け・法人向けいずれも、 「心のバランスを整える実践」に焦点をあてた内容です。

S

セミナー詳細

受講対象を明確にしたテーマ別セミナー。対面・オンライン双方に対応し、 ワーク中心で理解を深めます。

  • 対象：ストレス対策を学びたい方
  • 形式：グループ / 少人数制
  • 効果：感情整理・行動改善の実践力
セミナー詳細へ
C

カウンセリング詳細

個別対話で課題を整理し、生活や仕事の中で再現できる具体的なセルフケア計画を設計します。

  • 対象：継続的に伴走がほしい方
  • 形式：1対1（対面 / オンライン）
  • 効果：思考の癖の可視化と調整
カウンセリング詳細へ
B

企業向けサービス

管理職・チーム向けのメンタルヘルス研修。現場課題に合わせてカスタマイズし、 稟議用資料の提供にも対応します。

  • 対象：企業・自治体・教育機関
  • 形式：講演 / 研修 / 伴走支援
  • 効果：心理的安全性と対話品質の向上
