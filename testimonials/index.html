カテゴリ別のお客様の声

有銘宏司さん / 体感講座 / 2025年12月

「途中参加でも深い気づきがあった」

途中からの参加でしたが、才能を活かすための自己理解が一気に深まりました。ありがとうございました。

M.Y.さん / カウンセリング / 2025年11月

「カウンセリング後に現実が急展開した」

恵子先生のカウンセリングを申し込んでほんの数日後、事態は動き出していました。最初に答えを教えていただき「夫とタッグを組む」。止まっていた工事が急展開し、注文が入ってこないと思っていたお客様たちからもダダダーっと注文をいただくということが起きました。こんな事あるんだ、という気持ちです。

山田明弘さん / 体感講座 / 2025年9月

「相手のマイナスは自分のマイナスだった」

人と人とは、どちらかが優れている訳でもなく劣っている訳でもなく、鏡の如くお互いにプラス要素とマイナス要素がある。まさか両親を嫌っていたことは自分を嫌っていたことだったとは。理想やプラス思考、被害者意識、満載の人生でした。思考が作った間違った自己イメージを解放していくことができるようになってきました。

K.N.さん / 集中講座 / 2025年7月

「自分が自分に対して支配的だった」

自分が自分に対して威圧的で、常識という枠を押し付け、常に無視していた。感情の苦しみ、悲しみはあってはならないものとして隠して無いものにしようとしていた。息も吸えないような厳しさがあるから常に依存者を目の前に作り出してきました。その気づきが得られた集中講座でした。

Y.S.さん / Lifeコース / 2025年6月

「理想を手放せない自分に気づいた」

理想を大きく持てば持つほどコンプレックスも大きくなる――それでもやっぱり理想を追い求めていました。あの空間には愛が溢れていて、お話を聞いているだけで頑なな自分に気づき、ゆるんでいくのがわかりました。恵子先生の主導でどんどん愛の空間が創造されていき、あっという間でしたが中身はとっても濃くて一番変化を感じたLIFEコースでした。

M.K.さん / 体感講座 / 2025年5月

「母を愛おしく感じることができた」

元々持っているプログラムが発動され、間違った自己イメージが強化され、パターン化のトライアングルにはまり込んで抜け出せなくなる。今回はとても愛情あふれる母として全身全霊で生きている、頼もしくも健気で愛おしく感じることができました。母が大好きすぎて途中から嫌いになればこんな思いをしなくていいんだと、子どもながらに実行していたことを思い出しました。

太田恵さん / カウンセリング / 2025年3月

「両親を選んで生まれてきた実感」

楽しい中にも的確にナビゲートしてくださり癒しの時間でした。今回の出来事から私の小さい頃の思い、今の思い、両親の思い、そして自分の子どもの思いまで感じることができました。こんなこと絶対に３次元ではわからないこと。わたしはこれを知りたくて両親を選んで生まれてきた。実感できました。

石井広之さん / Lifeコース / 2024年10月

「抑え込んでた怒りを出せた」

先週からあっと言う間の1週間でした。今日は前回抑え込んでた怒りを出して下さった仲間への感謝の気持ちでの始まりでした。

E.M.さん / 体感講座 / 2024年5月

「"ダメ人間"のレッテルから解放された」

「自分という理解を変えていく」というレクチャーから、間違った自己イメージからまっさらな状態に持っていくことだと受け取らせていただき、"ダメ人間"という自分のレッテルから解放され、肩の緊張が解ける感じでした。

K.O.さん / カウンセリング / 2024年1月

「364回目の記念すべきカウンセリング」

今までなら有耶無耶にして放っておいた感情を、カウンセリングの空間で出させていただいたことで、自分の変化を改めてしっかりと感じることができました。

原真由美さん / セミナー

「楽しい記憶が弱い理由がわかった」

悲しみを感じたくなくて小さくしたことで、喜びも小さくなった。この表裏一体にびっくりしました。幼少時代からの感情の切り方を、このセミナーで知ることができました。
